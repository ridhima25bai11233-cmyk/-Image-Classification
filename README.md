# -Image-Classification
I did a project on CIFAR-10 Image Classification using All-CNN. This was a Deep Learning project that I worked on as part of my Artificial Intelligence and Machine Learning classes.
Project Overview
This project implements an All-Convolutional Neural Network (All-CNN) to classify images from the CIFAR-10 dataset into 10 categories: Airplane, Automobile, Bird, Cat, Deer, Dog, Frog, Horse, Ship, and Truck.

The All-CNN architecture replaces traditional fully connected layers with convolutional layers, making it efficient and well-suited for image classification tasks.

Tech Stack

Results
The model achieved approximately 90% accuracy on the test dataset, demonstrating strong performance in classifying images across all categories.
